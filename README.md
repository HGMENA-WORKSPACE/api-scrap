# Scrap Server

This is a small scrap server for apis.

## Description

In order for the end user to approve your application to access its data and functions, they must provide an api url, a start and end of scrap in the .env file.


## Getting Started

### Dependencies

* node

### Installing

```
  npm i
```


### Executing program

```
  npm start
```



## Lint

```
  npm run lint
```
